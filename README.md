# OTT Trusted - Premium Dashboard

A sleek, modern OTT subscription management platform built with React, Vite, and Supabase.

## Features
- **Premium Catalog**: Access to high-quality activation slots for major platforms.
- **Instant Activation**: Automated profile delivery.
- **Secure Payments**: Integrated payment protocols.
- **Admin Dashboard**: Full control over app listings and orders.

## Tech Stack
- **Frontend**: React 19, Tailwind CSS v4, Framer Motion
- **Backend/Auth**: Supabase
- **Icons**: Lucide React
- **Build Tool**: Vite
